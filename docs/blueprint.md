# **App Name**: Mega Hacker Virada

## Core Features:

- Number Generation: Generate a set of numbers for Mega da Virada based on statistical analysis and AI.
- Payment Processing: Securely process user payments before generating numbers.  Each generation requires a new payment.
- Results Display: Display the generated numbers with a neon highlight effect, emphasizing their exclusivity.
- AI Analysis Animation: Show a progress animation of 'IA analysing secret database' and 'hacking server' before number generation.
- Admin Panel: Admin panel for seeing user requests, number generations, and user purchase analytics
- Exclusive Messaging: Provide messages like 'Validating Request', 'Access Granted', 'Canal Confidencial - Apenas Membros'

## Style Guidelines:

- Primary color: Steel gray (#D7D7D7) to provide a technological feel with a hint of light.
- Background color: Pure black (#000000) to convey a sense of secrecy and a dark web aesthetic.
- Accent color: Alarming red (#FF2E2E) for interactive elements, errors and alerts, providing an aggressive touch.
- Body and headline font: 'Source Code Pro' (monospace) for a code infiltration look.
- Logo incorporating a hacker mask, the number '2025,' and green neon code elements with a data leak appearance.
- Animated logo appearing to assemble itself from encrypted pieces within 2 seconds.
- Light text animations imitating a terminal typing effect for titles and subtitles.
- Full-screen layout on the home page with animated scrolling code background to give the feeling of a forbidden terminal.
